todo-sails-backbone

a sample Sails+Backbone+Mongo application

Instructions to run application:

  1. Clone the repo
  2. "cd" into the folder - checkout db-mongo branch
  3. npm install
  4. edit the mongo details in config/adapters.js
  5. sails lift
